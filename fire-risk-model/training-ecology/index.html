    Stage 1 — Ecology Model Training¶

    The ecology model is a multi-head U-Net that predicts four per-pixel quantities simultaneously from spectral indices computed on Red / NIR / RedEdge bands.

    Architecture¶

    Input: raw bands (B, 3, H, W)
    ↓ IndicesLayer
5-channel spectral indices (B, 5, H, W)
    NDVI · NDRE · RRI2 · CIre · NIR_texture
    ↓ _EcologyEncoder (4-scale U-Net encoder)
    ↓ 4 independent decoder heads
┌─────────────────────────────────────────────────────┐
│  landcover   → (B, 20, H, W)  cross-entropy         │
│  tsd         → (B,  1, H, W)  regression (Huber)    │
│  crown_close → (B,  1, H, W)  regression (Huber)    │
│  species     → (B,  5, H, W)  regression (Huber)    │
└─────────────────────────────────────────────────────┘

    Embedding dimension: 28 channels (ECOLOGY_EMBED_DIM_FULL)
    = 20 (landcover probs) + 1 (TSD) + 1 (crown closure) + 5 (species) + 1 (fuel density)

    Training command¶

    python scripts/train_ecology.py --train \
    --data-dir   data/tiles \
    --epochs     50 \
    --base-ch    32 \
    --out-dir    experiments/ecology

    Key flags¶

    Flag Default Description
    --epochs 50 Training epochs
    --lr 1e-3 Initial learning rate (cosine decay)
    --base-ch 32 U-Net base channel width
    --batch-size 8 Batch size
    --num-workers 0 DataLoader worker processes
    --tile-size 256 Spatial crop size during training
    --weights — Resume from / fine-tune a checkpoint

    Loss weights¶

    The multi-task loss is a weighted sum across the four heads:

    Flag Default Head
    --w-landcover 0.30 Landcover cross-entropy
    --w-tsd 0.15 Time-since-disturbance Huber loss
    --w-crown-close 0.10 Crown closure Huber loss
    --w-species 0.30 Species composition Huber loss

    Checkpoint selection¶

    The best checkpoint is saved when lcc_miou + spp_mean_r2 improves on the validation set. Training prints a metrics table at the end of each epoch:

    Epoch 12/50
  train  loss=0.412  lcc_miou=0.521  tsd_mae=0.043  cc_r2=0.671  spp_r2=0.589
  val    loss=0.438  lcc_miou=0.498  tsd_mae=0.051  cc_r2=0.643  spp_r2=0.561
  ✓ New best  (lcc_miou + spp_r2 = 1.059)  → experiments/ecology/ecology_best.pt

    Evaluation¶

    python scripts/train_ecology.py --eval \
    --data-dir data/tiles \
    --weights  experiments/ecology/ecology_best.pt

    Uncertainty quantification¶

    The trained model supports MC-Dropout uncertainty estimation:

    from src.models.ecology import EcologyModel

model = EcologyModel()
model.load("experiments/ecology/ecology_best.pt")

# Returns (mean_embed, std_embed) — both (B, 28, H, W)
mean, std = model.embed_with_uncertainty(x, n_passes=20)

# Concatenate for AugmentedFireRiskModel with use_uncertainty=True
import torch
ecology = torch.cat([mean, std], dim=1)   # (B, 56, H, W)

    Preliminary training

    For a quick diagnostic run (5–15 epochs), the training curve and LCC weight printout show whether all 20 classes are represented and whether each head is learning. A full 50-epoch run is recommended before moving to Stage 2.
