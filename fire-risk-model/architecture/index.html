    Architecture¶

    IndicesLayer¶

    src/models/indices.py

    A differentiable PyTorch layer that converts raw band tensors into spectral index tensors. Runs inside both Stage 1 and Stage 2 so gradients flow back through it.

    # Input:  (B, n_bands, H, W)  raw reflectance values
# Output: (B, 5, H, W)        normalised spectral indices
indices = IndicesLayer(
    red_idx     = 0,
    nir_idx     = 2,
    rededge_idx = 1,
    sensor_mask = [1, 1, 1, 1, 1],   # all 5 indices active
)
    Index Channels out Formula Notes
    NDVI 1 (NIR − Red) / (NIR + Red) Clamped to [−1, 1]
    NDRE 1 (NIR − RE) / (NIR + RE) Clamped to [−1, 1]
    RRI2 1 Red / (RE² + ε) Clamped to [0, 1]
    CIre 1 NIR / (RE + ε) − 1, ÷ 10 Sensitive to high LAI; less saturation than NDVI
    NIR_texture 1 Local std of NIR (15-px window) ÷ 0.3 Crown roughness proxy

    sensor_mask zeros out indices that cannot be computed from available bands (e.g. sentinel2 profile disables CIre and NIR_texture).

    When use_raw_bands=True, the 3 raw band values are concatenated after the indices → 8-channel output.

    EcologyModel (Stage 1)¶

    src/models/ecology.py

    Multi-head U-Net encoder-decoder. Shared encoder, independent decoder per head.

    _EcologyEncoder
  in_channels = 5  (IndicesLayer output)
  base_ch     = 32
  4 encoder stages: 32 → 64 → 128 → 256 channels
  dilation bottleneck

  → shared feature map (B, 256, H/8, W/8)

_EcologyDecoder  ×4 (one per head)
  Skip connections from encoder stages
  Head outputs at full resolution (B, C, H, W)

    Forward output dict¶

    out = model(x)   # x: (B, 3, H, W) raw bands

out["landcover"]   # (B, 20, H, W)  logits — apply softmax for probabilities
out["tsd"]         # (B,  1, H, W)  [0, 1]
out["crown_close"] # (B,  1, H, W)  [0, 1]
out["species"]     # (B,  5, H, W)  [0, 1] per species

    Embedding methods¶

    # 28-channel embedding for Stage 2 (no uncertainty)
embed = model.embed_full(x)            # (B, 28, H, W)

# 28-channel mean + 28-channel std from MC-Dropout
mean, std = model.embed_with_uncertainty(x, n_passes=20)

# Pass to AugmentedFireRiskModel with use_uncertainty=True
import torch
ecology = torch.cat([mean, std], dim=1)   # (B, 56, H, W)

    Embedding channel layout (28 channels)¶

    Channels Content
    0–19 Landcover class probabilities (softmax)
    20 Time-since-disturbance (TSD)
    21 Crown closure
    22–26 Species fractions (Pjack, Pblack, Balsam, Poplar, Spruce)
    27 Fuel density = crown_close × Σ(species × flammability_weights)

    Fuel density¶

    EcologyModel.fuel_density(out) computes a single-channel canopy fuel proxy:

    fuel_density = crown_close × Σ(species_probs × [0.9, 0.9, 0.7, 0.3, 0.8])

    Flammability weights: jack pine (0.9) · black spruce (0.9) · balsam fir (0.7) · broadleaf (0.3) · white/red pine (0.8).

    AugmentedFireRiskModel (Stage 2)¶

    src/models/cnn.py

    model = AugmentedFireRiskModel(
    ecology_dim     = 28,       # ECOLOGY_EMBED_DIM_FULL
    use_uncertainty = False,    # set True to accept cat([mean, std])
    sensor_profile  = "full",
    base_ch         = 32,
    dropout         = 0.3,
)
# Total input channels: 5 (indices) + 28 (ecology) = 33

    With uncertainty:

    model = AugmentedFireRiskModel(
    ecology_dim     = 28,
    use_uncertainty = True,    # expects 56-channel ecology input
)
# Total input channels: 5 + 56 = 61

    Physics-seeded filters¶

    The first encoder convolution is seeded with 14 hand-crafted filters targeting fire-relevant spectral transitions (center-surround and Sobel on all 5 index channels). The remaining 18 filters learn freely from data.

    FireRiskModel (Stage 2, standalone)¶

    For single-stage baselines without an ecology embedding:

    from src.models.cnn import FireRiskModel

model = FireRiskModel(
    red_idx        = 0,
    nir_idx        = 2,
    rededge_idx    = 1,
    sensor_profile = "full",
    model_variant  = "unet",   # or "plain_encdec"
)

    Foundation model adapters¶

    Both adapters accept the same 5-channel IndicesLayer output and return a per-pixel binary segmentation map, making them drop-in replacements for FireRiskCNN in the training loop.

    Model Parameters Notes
    Prithvi-100M ~100 M NASA / IBM geospatial foundation model
    SatMAE ViT-L ~307 M MAE pre-trained on satellite imagery 
    python scripts/train.py \
    --model-variant prithvi_segmentation \
    --data-dir      data/patches
