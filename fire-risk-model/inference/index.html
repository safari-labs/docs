    Inference¶

    Two-stage pipeline¶

    Both stages run in a single call. The ecology model produces a per-pixel embedding; the fire risk model consumes it along with the raw spectral indices.

    Single scene¶

    python scripts/infer_pipeline.py \
    --input            scene.tif \
    --bands            0 1 2 \
    --ecology-weights  experiments/ecology/ecology_best.pt \
    --fire-weights     experiments/fire/fire_best.pt \
    --out-dir          output/

    Batch directory¶

    python scripts/infer_pipeline.py \
    --input-dir        scenes/ \
    --bands            0 1 2 \
    --ecology-weights  experiments/ecology/ecology_best.pt \
    --fire-weights     experiments/fire/fire_best.pt \
    --out-dir          output/

    Output files¶

    For each input scene the pipeline writes:

    File Dtype Content
    *_ecology_lcc.tif uint8 Landcover class (0–19)
    *_ecology_tsd.tif float32 Time-since-disturbance [0, 1]
    *_ecology_cc.tif float32 Crown closure [0, 1]
    *_ecology_fuel.tif float32 Fuel density proxy [0, 1]
    *_ecology_spp_Pjack.tif float32 Jack pine fraction
    *_ecology_spp_Pblack.tif float32 Black spruce fraction
    *_ecology_spp_Balsam.tif float32 Balsam fir fraction
    *_ecology_spp_Poplar.tif float32 Broadleaf fraction
    *_ecology_spp_Spruce.tif float32 White/red pine fraction
    *_fire_risk.tif float32 Fire risk probability [0, 1]
    *_fire_risk_class.tif uint8 0 = Low / 1 = Medium / 2 = High
    *_pipeline.png — Composite visualisation

    Fire risk thresholds: Low < 0.3 · Medium 0.3–0.6 · High > 0.6

    With uncertainty¶

    python scripts/infer_pipeline.py \
    --input           scene.tif \
    --bands           0 1 2 \
    --ecology-weights experiments/ecology/ecology_best.pt \
    --fire-weights    experiments/fire_unc/fire_best.pt \
    --use-uncertainty \
    --mc-passes       20 \
    --out-dir         output/

    Additional outputs when --use-uncertainty is active:

    File Content
    *_ecology_lcc_std.tif Per-class std of landcover probs across MC passes
    *_ecology_tsd_std.tif TSD uncertainty
    *_ecology_cc_std.tif Crown closure uncertainty
    *_fire_risk_std.tif Fire risk map standard deviation

    Reflection padding

    All inference calls use predict() with 32-pixel reflection padding to suppress edge artifacts at tile boundaries. The padding is cropped from the output before saving.

    Mission-time validation¶

    Compare ecology model predictions against SCANFI v2 on field-acquired GeoTIFFs:

    python scripts/validate_ecology.py \
    --input-dir  mission/scenes/ \
    --scanfi-dir data/scanfi \
    --weights    experiments/ecology/ecology_best.pt \
    --out-dir    output/validation

    Outputs: validation_summary.json · per_scene_metrics.csv · validation_summary.png + per-scene diagnostic plots (confusion matrix, TSD scatter, species radar).
