    Stage 2 — Fire Risk Model Training¶

    The fire risk model fuses the ecology embedding from Stage 1 with spectral indices to produce a pixel-wise fire risk probability.

    Architecture¶

    Raw bands (B, 3, H, W)
    ↓ IndicesLayer
Spectral indices (B, 5, H, W)
    ↓
    ├── EcologyModel.embed_full()  →  ecology embedding (B, 28, H, W)
    │   [or embed_with_uncertainty() →  cat([mean, std]) (B, 56, H, W)]
    │
    └── concat ──────────────────────────────────────────────
                                                             ↓
                                           AugmentedFireRiskModel
                                           (5 + 28 = 33 input channels)
                                                             ↓
                                           fire risk map [0, 1]  (B, 1, H, W)

    Training command¶

    python scripts/train.py \
    --data-dir         data/patches \
    --ecology-weights  experiments/ecology/ecology_best.pt \
    --out-dir          experiments/fire \
    --epochs           60

    With uncertainty embedding¶

    python scripts/train.py \
    --data-dir         data/patches \
    --ecology-weights  experiments/ecology/ecology_best.pt \
    --use-uncertainty \
    --out-dir          experiments/fire_unc \
    --epochs           60

    When --use-uncertainty is set, the Stage 2 model receives cat([mean_embed, std_embed]) — 56 channels instead of 28 — giving it access to per-pixel ecology confidence for each prediction.

    Physics-seeded initialisation¶

    The first encoder layer is seeded with hand-crafted filters before training:

    Filter Channel Pattern Detects
    0 NDVI Center-surround (LOW) Fire scar boundary in healthy canopy
    1 NDRE Center-surround (LOW) Chlorophyll stress boundary
    2–3 NDVI Sobel X/Y NDVI edges
    4–5 NDRE Sobel X/Y Stress edges
    6–7 RRI2 Sobel X/Y Canopy structure edges
    8 CIre Center-surround (LOW) Chlorophyll gap / canopy stress
    9–10 CIre Sobel X/Y Active-fire boundary into healthy stand
    11 NIR_texture Center-surround (HIGH) Rough conifer patch
    12–13 NIR_texture Sobel X/Y Conifer / broadleaf transition

    Remaining filters initialise with Kaiming normal and learn freely.

    Model variants¶

    Variant Flag Notes
    U-Net (default) --model unet Skip connections, physics-seeded enc1
    Plain enc-dec --model plain_encdec No skip connections (ablation)
    DeepLabV3+ ResNet-50 --model-variant deeplabv3plus_resnet50 SMP baseline
    U-Net EfficientNet-B0 --model-variant unet_efficientnet_b0 SMP baseline
    Prithvi-100M --model-variant prithvi_segmentation Foundation model
    SatMAE ViT-L --model-variant satmae_segmentation Foundation model

    Baseline / ablation experiments¶

    # Reproduce all baselines (3 independent runs each)
python scripts/run_experiments.py --exps baseline --n-runs 3

# Regularisation ablation
python scripts/run_experiments.py --exps regularization --n-runs 3

    Results are written to experiments/<run_name>/metrics.json.

    Sensor profile¶

    Use --sensor-profile to match the active sensor:

    Profile Bands active Use case
    full NDVI, NDRE, RRI2, CIre, NIR_texture Modified A7S II (R/RE/NIR)
    sentinel2 NDVI, NDRE, RRI2 Sentinel-2 Band 4/8/8A
    micasense NDVI, NDRE, RRI2 MicaSense RedEdge
    a7sii NDVI, NDRE, RRI2, CIre, NIR_texture Sony A7S II (IR-cut removed)

    Checkpoint compatibility

    Changing --sensor-profile or the number of spectral indices changes IndicesLayer.n_out and therefore _EcologyEncoder.in_channels. Existing checkpoints trained with a different profile cannot be reloaded — a full retrain is required.
