    Subsystem — Altimeter MS5607¶

    Sensor: MS5607-02BA03-50 — TE Connectivity
    Measures: Pressure (10–1200 hPa) · Temperature (−40…+85 °C) · Altitude (m)
    Hardware target: SPI on GPIO8–11
    Current firmware: I2C via bundled MS5607 library (Libnexamples/MS5607-master/)
    Files: include/altimeter.h · src/altimeter.cpp

    1. Hardware Connections¶

    @startuml
skinparam componentStyle rectangle

node "Raspberry Pi Pico (RP2040)" {
    component "GPIO8  (SPI MISO / SDO)" as MISO
    component "GPIO9  (SPI CS  / CSB)"  as CS
    component "GPIO10 (SPI SCK / SCLK)" as SCK
    component "GPIO11 (SPI MOSI / SDI)" as MOSI
}

component "MS5607\n(pressure + temp)" as MS {
    component "ADC\n24-bit" as ADC_MS
    component "PROM\n(calibration\ncoefficients C1-C6)" as PROM
}

MISO <-- MS  : SDO (sensor → MCU)
CS   --> MS  : CSB (active LOW)
SCK  --> MS  : SCLK
MOSI --> MS  : SDI (MCU → sensor)

note bottom of MS
  Current firmware uses I2C (library limitation).
  SPI pins reserved on PCB.
  Replace _sensor driver in altimeter.h/.cpp
  when an SPI MS5607 library is available.
end note
@enduml
    GPIO SPI signal MS5607 pin Status
    GPIO8 MISO SDO Reserved — not driven by current firmware
    GPIO9 CS CSB Reserved — not driven by current firmware
    GPIO10 SCK SCLK Reserved — not driven by current firmware
    GPIO11 MOSI SDI Reserved — not driven by current firmware

    3. Initialization Sequence — init()¶

    @startuml
participant AltimeterModule as alt
participant MS5607          as lib

alt -> lib : _sensor.begin()
note right : reads PROM calibration\ncoefficients C1-C6 via I2C

alt lib found
    lib --> alt : true
    alt -> alt  : _ready = true\n_error = OK
    alt -> alt  : Serial.println("[Altimeter] OK")
else not found
    lib --> alt : false
    alt -> alt  : _error = NOT_FOUND\n_ready = false
    alt -> alt  : Serial.println("[Altimeter] not found")
end
@enduml

    4. Measurement Sequence — read()¶

    @startuml
participant AltimeterModule  as alt
participant MS5607            as lib
participant "g_adcs.altimeter" as data

alt -> lib : _sensor.readDigitalValue()
note right of lib
  I2C transactions:
  · trigger D1 (pressure ADC)  → wait
  · read D1 (3 bytes)
  · trigger D2 (temp ADC)      → wait
  · read D2 (3 bytes)
  · 2nd-order compensation
  NOP library ≈ 36 ops
end note

alt lib OK
    lib --> alt : true
    alt -> lib  : getTemperature() → T
    alt -> lib  : getPressure()    → P
    alt -> lib  : getAltitude()    → alt_m
    note right : getAltitude uses pow()\nNOP ≈ 23 ops

    alt T and P in valid range
        alt -> data : pressure    = P    [hPa]
        alt -> data : temperature = T    [°C]
        alt -> data : altitude    = alt_m [m]
        alt -> data : valid = true
        alt -> alt  : _failCount = 0\n_error = OK
        alt --> alt : return true
    else out of range
        alt -> data : valid = false
        alt -> alt  : _error = DATA_INVALID
        alt --> alt : return false
    end

else read failed
    lib --> alt  : false
    alt  -> data : valid = false
    alt  -> alt  : _error = READ_FAIL\n_failCount++
    alt  --> alt : return false
end
@enduml

    5. MS5607 Compensation Formulas¶

    @startuml
start

:Read raw values D1 (pressure), D2 (temperature)
 from ADC via I2C;

:Read PROM coefficients C1…C6
 (done once in begin());

partition "Temperature compensation" {
    :dT   = D2 − C5 × 2^8;
    :TEMP = 2000 + dT × C6 / 2^23;
}

partition "Pressure compensation" {
    :OFF  = C2 × 2^17 + (C4 × dT) / 2^6;
    :SENS = C1 × 2^16 + (C3 × dT) / 2^7;
    :P    = (D1 × SENS / 2^21 − OFF) / 2^15;
}

if (TEMP < 2000  [T < 20 °C]?) then (yes)
    partition "2nd-order correction" {
        :T2    = dT^2 / 2^31;
        :OFF2  = 61 × (TEMP−2000)^2 / 2^4;
        :SENS2 = 2  × (TEMP−2000)^2;
        :apply corrections to TEMP, OFF, SENS, P;
    }
endif

:getAltitude():
 alt = 44330 × (1 − (P/101325)^0.190295);

stop
@enduml

    6. Validation Ranges (firmware-enforced)¶

    @startuml
start

:P = getPressure()
T = getTemperature();

if (P < 10 or P > 1200 hPa\nor T < -40 or T > 85 °C?) then (yes)
    :_error = DATA_INVALID
valid = false;
    stop
else (no)
    :write P, T, alt to g_adcs
valid = true;
    stop
endif
@enduml
    Field Valid range
    Pressure 10 – 1200 hPa
    Temperature −40 °C – +85 °C
    Altitude derived — not independently validated
