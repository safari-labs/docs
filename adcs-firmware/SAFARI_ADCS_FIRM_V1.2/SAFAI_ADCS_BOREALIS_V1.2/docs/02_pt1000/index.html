    Subsystem — Pt1000 Temperature Sensors¶

    Sensor: PTFC102B1G0 — TE Connectivity (PTF family, Class B)
    Channels: 8 × Pt1000, analog 8:1 MUX
    Files: include/pt1000.h · src/pt1000.cpp

    1. Hardware Connections¶

    @startuml
skinparam componentStyle rectangle

node "Raspberry Pi Pico (RP2040)" {
    component "GPIO0  (MUX A0)" as A0
    component "GPIO1  (MUX A1)" as A1
    component "GPIO2  (MUX A2)" as A2
    component "GPIO3  (SW)"     as SW
    component "GPIO26 (ADC0)"   as ADC
}

component "MUX 8:1\n(CD4051 or equiv.)" as MUX
component "Op-amp\n(voltage buffer)"     as AMP
component "S1…S8\nPt1000 (1000 Ω @ 0°C)" as PT

A0  --> MUX : A0
A1  --> MUX : A1
A2  --> MUX : A2
SW  --> MUX : EN / power switch

MUX --> PT  : selects one channel
PT  --> AMP : Vpt1000
AMP --> ADC : Vout  [0 … 3.3 V]
@enduml
    GPIO Role State
    GPIO0–2 MUX address A2:A1:A0 OUTPUT
    GPIO3 Circuit power switch HIGH = ON, LOW = OFF
    GPIO26 ADC0 — op-amp output INPUT ANALOG

    3. Initialization Sequence — init()¶

    @startuml
participant "Pt1000Module" as pt
participant "Arduino GPIO" as gpio
participant "Arduino ADC"  as adc

pt -> gpio : pinMode(A0,A1,A2,SW, OUTPUT)
pt -> gpio : digitalWrite(SW, LOW)   ' circuit OFF
pt -> adc  : analogReadResolution(10)

note over pt : Smoke test — confirm ADC responds
pt -> gpio : selectChannel(0)
pt -> gpio : digitalWrite(SW, HIGH)  ' circuit ON
pt -> pt   : delay(5 ms)
pt -> adc  : analogRead(GPIO26) → raw  [discarded]
pt -> gpio : digitalWrite(SW, LOW)   ' circuit OFF

pt -> pt   : _ready = true\n_error = OK
pt -> pt   : Serial.println("[Pt1000] OK")
@enduml

    4. Measurement Sequence — read() (8 channels)¶

    @startuml
participant "Pt1000Module" as pt
participant "MUX (GPIO0-2)" as mux
participant "ADC (GPIO26)"  as adc
participant "g_adcs.pt1000" as data

pt -> pt  : digitalWrite(SW, HIGH)  ' power ON

loop ch = 0 to 7

    pt  -> mux : selectChannel(ch)\n  A0=(ch>>0)&1\n  A1=(ch>>1)&1\n  A2=(ch>>2)&1
    pt  -> pt  : delayMicroseconds(500)  ' RC settling

    pt  -> adc : analogRead(GPIO26) → raw
    pt  -> pt  : v = raw / 1023.0 × 3.3

    alt v in [LUT_V[0], LUT_V[199]]
        pt -> pt   : T = lutLookup(v)
        pt -> data : temps[ch] = T\nvalid[ch] = true\nactive_count++
    else out of range
        pt -> data : temps[ch] = 0.0\nvalid[ch] = false
    end

end

pt -> pt : digitalWrite(SW, LOW)  ' power OFF
@enduml

    5. LUT Binary-Search Algorithm — lutLookup(v)¶

    @startuml
start

:lo = 0
hi = 199;

while (hi − lo > 1?) is (yes)
    :mid = lo + (hi − lo) / 2;
    if (LUT_V[mid] ≤ v?) then (yes)
        :lo = mid;
    else (no)
        :hi = mid;
    endif
endwhile

:return T_CAL_MIN + lo × T_CAL_STEP;

stop

note right
  NOP ≈ 58 (7 iterations × 8 ops + 2)
  Resolution: ~0.7035 °C/step (floor)
  No interpolation applied
end note
@enduml

    Complexity:

    Metric Value
    Time O(log N) — 7 iterations for N = 200
    NOP ≈ 58 ops per call
    Stack 10 B (lo, hi, mid, v)
    Flash 800 B (LUT_V[200])
