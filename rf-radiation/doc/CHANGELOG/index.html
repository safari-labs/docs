    @app.route("/api/link_budget")
def api_link_budget():
    tx = request.args.get("tx", "dipole_half_wave")
    rx = request.args.get("rx", "yagi")

    # Delegate to controller
    budget, status = link_budget_controller.calculate_link_budget(
        tx=tx, rx=rx,
        gs_lat=GROUND_STATION["lat"],
        gs_lon=GROUND_STATION["lon"],
        gs_alt=GROUND_STATION["alt"]

    if status != 200:
        return jsonify(budget), status

    Testing Improvements¶

    Before (Hard to test)¶

    # To test antenna pattern:
# - Had to import server_2.py (imports Flask, CORS, loads files)
# - Had to mock Flask context
# - Had to set up entire app
# - Could only test in integration

# To test RF calculation:
# - Had to import server_2.py
# - Had to find the calculation buried in 973 lines
# - Had to mock all dependencies
# - Very brittle

    After (Easy to test)¶

    # To test antenna pattern:

    Development Workflow Changes¶

    Before¶

    1. Edit server_2.py
    2. Find related code (grep entire file)
    3. Test everything (risk of breakage)
    4. Deploy monolithic file

    After¶

    1. Identify which component to edit (models, controllers, services)
    2. Edit focused file (~100-300 lines instead of 973)
    3. Test that component (isolated unit test)
    4. Deploy individual files + routes

    Time to add feature: ~3 hours → ~1 hour Risk of breaking something: High → Low
